    Beethoven.Allocator (Beethoven v0.3.7)

    Allocator is a stack of processes that facilitate the aggregation of telemetry signals to determine how busy a given Beethoven node is.

    Public API

    • allocate/0 Provides the URI of the least-busy node in the cluster.

    The busyness of nodes is determined via signals. Refer to the documentation for Beethoven.Allocator.Agent for more information on creating signals.
